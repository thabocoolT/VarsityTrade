# Varsity Trade

Varsity Trade is a full-stack student marketplace web application built for South African universities. It allows students to buy, sell, and trade items safely within their own campus — every listing is locked to the seller's registered university, so buyers only ever see items from students at their own institution.

## Built With

- ASP.NET Core Web API — backend REST API
- ASP.NET MVC with Razor — frontend web application
- SQL Server + Entity Framework Core — database and ORM
- .NET MAUI — mobile application (Phase 8)

## Features

- University-locked marketplace for campus safety
- Dual-role system — every user registers as a Buyer and can optionally activate a Seller profile
- In-app messaging between buyers and sellers
- Trade and cash offer system
- Reviews gated by completed transactions
- Admin panel for moderation, reporting, and platform management
- Hero banner management for platform announcements
- JWT authentication with refresh token support
